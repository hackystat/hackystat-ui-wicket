ProjectDatePanel


  • Unit Test



When completed, this page should contain a set of controls that allow a user to select a date, a project, and a kind of daily analysis to display. The resulting display could provide things like the following:

  • A 'snapshot' of the project with respect to that daily analysis on that day.
  • A comparison of that data to the state of the project yesterday, last week, or last month.
  • A 'table' and 'graph' checkbox, which enables the user to see the data in tabular or graphical form (using, for example google charts.)
